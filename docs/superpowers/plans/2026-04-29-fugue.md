# Fugue Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Native-AOT CLI-агент `fugue` на F# + Microsoft Agent Framework + Terminal.Gui.Elmish — лёгкая альтернатива Claude Code с TUI, мульти-провайдером и базовыми tools.

**Architecture:** Solution из 4 F#-проектов (Core/Agent/Tools/Cli) + xUnit-тестов. UI на Elmish-MVU, conversation-loop ручной поверх `Microsoft.Extensions.AI.IChatClient`, tools — `AIFunction` через `AIFunctionFactory.Create`. Конфиг — env + `~/.fugue/config.json` + discovery (env / `~/.claude/settings.json` / Ollama-ping).

**Tech Stack:** F# 9, .NET 10 (Native AOT), **Terminal.Gui v2.x** (без сторонних wrapper'ов), **свой mini-MVU модуль `Fugue.Cli.Mvu`** (~150 строк, заменяет Terminal.Gui.Elmish), Microsoft.Extensions.AI 10.5, **Microsoft.Agents.AI** + **Microsoft.Agents.AI.Anthropic** (first-party Microsoft Agent Framework 1.0), Microsoft.Extensions.AI.OpenAI, OllamaSharp 5.4, xUnit + FsUnit.

## Architecture decisions (locked after brainstorming + research iterations)

1. **Conversation loop — `AIAgent.RunStreamingAsync` (option A).** Идём через official Microsoft Agent Framework. Tool-call видимость в UI получаем парсингом `FunctionCallContent` / `FunctionResultContent` из `AgentResponseUpdate`-стрима. Бесплатно получаем: parallel tool-calls, retry/rate-limit, провайдер-нормализацию, AgentSession для истории, AIAgentMiddleware для permission-prompt в v0.2, MCP-клиент в v0.4.
2. **TUI — Terminal.Gui v2 + собственный mini-MVU (option B).** Один файл `src/Fugue.Cli/Mvu.fs` (~150 строк) даёт нам `Mvu.run init update view`, `Cmd.ofSub`, ленивый diff'инг детей View. AOT-clean by construction. Никаких внешних F#-wrapper'ов TG.
3. **Provider stack:** Anthropic — через `Microsoft.Agents.AI.Anthropic.AnthropicClient.AsAIAgent`. OpenAI — через `OpenAI.AsAIAgent` extension. Ollama — `OllamaApiClient.AsAIAgent` (через MAF extension над `IChatClient`). Все три нормализованы к `AIAgent`.

**Spec:** `docs/superpowers/specs/2026-04-29-fugue-design.md`

open System

/// Кто автор сообщения. System и Tool — служебные (служебные не показываем в UI).
type Role =
    | User
    | Assistant
    | System
    | Tool

/// Кусок сообщения. Один Message может содержать несколько ContentBlock'ов
/// (например, ассистент сначала пишет текст, потом вызывает tool).
type ContentBlock =
    | Text of string
    | ToolUse of id: string * name: string * argsJson: string
    | ToolResult of id: string * output: string * isError: bool

type Message =
    { Role: Role
      Blocks: ContentBlock list
      Timestamp: DateTimeOffset }

    static member text (role: Role) (text: string) =
        { Role = role
          Blocks = [ Text text ]
          Timestamp = DateTimeOffset.UtcNow }
```

module Fugue.Core.SystemPrompt

open System
open System.Runtime.InteropServices

/// Дефолтный system-prompt для Fugue. Подставляются cwd, OS, дата, доступные tools.
let render (cwd: string) (toolNames: string list) : string =
    let os =
        if RuntimeInformation.IsOSPlatform(OSPlatform.OSX) then "macOS"
        elif RuntimeInformation.IsOSPlatform(OSPlatform.Linux) then "Linux"
        elif RuntimeInformation.IsOSPlatform(OSPlatform.Windows) then "Windows"
        else "Unknown"

    let date = DateTime.UtcNow.ToString("yyyy-MM-dd")
    let tools = String.Join(", ", toolNames)

    $"""You are Fugue, a lean CLI coding assistant. You help the user with software engineering tasks in their terminal.

Working directory: {cwd}
Operating system: {os}
Today: {date}
Available tools: {tools}

# Tone
- Be concise. Default to short answers; expand only when asked.
- Prefer doing the task over describing it. Use tools.
- Never claim you ran a tool you did not run.

# Tools
- Read/Write/Edit work on text files. Edit requires an exact-match `old_string` that occurs once unless `replace_all` is set.
- Bash runs commands via `/bin/zsh -lc`. Default timeout is 120 seconds.
- Glob returns paths sorted by mtime descending.
- Grep returns matching lines with file:line prefixes.

# Style
- When you finish a task, stop. Do not summarize what you did unless asked.
- When you call a tool, do not also narrate "I will call X" — the user sees the call.
"""
```

```fsharp
module Fugue.Core.Config

open System
open System.IO
open System.Text.Json
open Fugue.Core.JsonContext

type ProviderConfig =
    | Anthropic of apiKey: string * model: string
    | OpenAI of apiKey: string * model: string
    | Ollama of endpoint: Uri * model: string

type AppConfig =
    { Provider: ProviderConfig
      SystemPrompt: string option   // None ⇒ use SystemPrompt.render at runtime
      MaxIterations: int }

type ConfigError =
    | NoConfigFound of help: string
    | InvalidConfig of reason: string

let private configPath () =
    let home = Environment.GetEnvironmentVariable("HOME") ?? Environment.GetEnvironmentVariable("USERPROFILE")
    Path.Combine(home, ".fugue", "config.json")

let private readEnv (name: string) =
    match Environment.GetEnvironmentVariable(name) with
    | null | "" -> None
    | v -> Some v

let private fromEnv () : ProviderConfig option =
    match readEnv "FUGUE_PROVIDER" with
    | Some "anthropic" ->
        readEnv "ANTHROPIC_API_KEY"
        |> Option.map (fun k ->
            let model = readEnv "FUGUE_MODEL" |> Option.defaultValue "claude-opus-4-7"
            Anthropic(k, model))
    | Some "openai" ->
        readEnv "OPENAI_API_KEY"
        |> Option.map (fun k ->
            let model = readEnv "FUGUE_MODEL" |> Option.defaultValue "gpt-5"
            OpenAI(k, model))
    | Some "ollama" ->
        let ep = readEnv "OLLAMA_ENDPOINT" |> Option.defaultValue "http://localhost:11434"
        let model = readEnv "FUGUE_MODEL" |> Option.defaultValue "llama3.1"
        Some(Ollama(Uri ep, model))
    | _ ->
        // Auto-detect by single-key presence
        match readEnv "ANTHROPIC_API_KEY", readEnv "OPENAI_API_KEY" with
        | Some k, _ -> Some(Anthropic(k, "claude-opus-4-7"))
        | _, Some k -> Some(OpenAI(k, "gpt-5"))
        | _ -> None

let private fromFile (path: string) : Result<ProviderConfig * int, string> =
    try
        let json = File.ReadAllText path
        let dto =
            JsonSerializer.Deserialize(
                json,
                AppConfigJsonContext.Default.AppConfigDto)
        let provider =
            match dto.provider with
            | "anthropic" -> Ok(Anthropic(dto.apiKey, dto.model))
            | "openai" -> Ok(OpenAI(dto.apiKey, dto.model))
            | "ollama" ->
                let ep = if String.IsNullOrEmpty dto.ollamaEndpoint then "http://localhost:11434" else dto.ollamaEndpoint
                Ok(Ollama(Uri ep, dto.model))
            | other -> Error $"unknown provider: {other}"
        provider |> Result.map (fun p -> p, dto.maxIterations)
    with ex -> Error ex.Message

let private helpText () =
    """No Fugue configuration found.

Set environment variables:
    FUGUE_PROVIDER=anthropic ANTHROPIC_API_KEY=sk-ant-... fugue
    FUGUE_PROVIDER=openai    OPENAI_API_KEY=sk-... fugue
    FUGUE_PROVIDER=ollama    OLLAMA_ENDPOINT=http://localhost:11434 fugue

…or create ~/.fugue/config.json (see docs).
"""

/// Загрузить конфиг. Discovery (для пустого случая) подключим в Task 22.
let load (_argv: string[]) : Result<AppConfig, ConfigError> =
    match fromEnv () with
    | Some provider ->
        Ok { Provider = provider; SystemPrompt = None; MaxIterations = 30 }
    | None ->
        let path = configPath ()
        if File.Exists path then
            match fromFile path with
            | Ok (p, mi) ->
                let mi = if mi <= 0 then 30 else mi
                Ok { Provider = p; SystemPrompt = None; MaxIterations = mi }
            | Error e -> Error(InvalidConfig e)
        else
            Error(NoConfigFound (helpText ()))

let saveToFile (cfg: AppConfig) : unit =
    let dir = Path.GetDirectoryName(configPath ())
    Directory.CreateDirectory(dir) |> ignore
    let dto =
        match cfg.Provider with
        | Anthropic(k, m) ->
            { provider = "anthropic"; model = m; apiKey = k; ollamaEndpoint = ""; maxIterations = cfg.MaxIterations }
        | OpenAI(k, m) ->
            { provider = "openai"; model = m; apiKey = k; ollamaEndpoint = ""; maxIterations = cfg.MaxIterations }
        | Ollama(e, m) ->
            { provider = "ollama"; model = m; apiKey = ""; ollamaEndpoint = string e; maxIterations = cfg.MaxIterations }
    let json =
        JsonSerializer.Serialize(
            dto,
            AppConfigJsonContext.Default.AppConfigDto)
    File.WriteAllText(configPath (), json)
```

module Fugue.Tools.PathSafety

open System
open System.IO

/// Canonicalize a user-supplied path. Relative paths anchor to cwd. Absolute paths kept.
let resolve (cwd: string) (path: string) : string =
    if Path.IsPathRooted path then
        Path.GetFullPath path
    else
        Path.GetFullPath(Path.Combine(cwd, path))

/// Returns true if `path` is contained in `root` (case-sensitive on Linux/macOS).
let isUnder (root: string) (path: string) : bool =
    let r = (Path.GetFullPath root).TrimEnd(Path.DirectorySeparatorChar) + string Path.DirectorySeparatorChar
    let p = Path.GetFullPath path
    p.StartsWith(r, StringComparison.Ordinal) || p = r.TrimEnd(Path.DirectorySeparatorChar)

module Fugue.Tools.ReadTool

open System
open System.IO
open System.ComponentModel
open Fugue.Tools.PathSafety

/// Read a text file, returning lines numbered cat -n style.
/// `offset` (1-based-equivalent) skips that many leading lines; `limit` caps line count.
[<Description("Read a text file. Returns lines prefixed with 1-based line numbers (cat -n format).")>]
let read
    ([<Description("File path (absolute or relative to working directory).")>] cwd: string)
    ([<Description("File to read.")>] path: string)
    ([<Description("Skip this many leading lines (default 0).")>] offset: int option)
    ([<Description("Read at most this many lines (default unlimited).")>] limit: int option)
    : string =
    let full = resolve cwd path
    if not (File.Exists full) then raise (FileNotFoundException("file not found", full))

    let allLines = File.ReadAllLines full
    let off = offset |> Option.defaultValue 0
    let take =
        match limit with
        | Some n -> n
        | None -> allLines.Length - off

    seq {
        for i in 0 .. take - 1 do
            let idx = off + i
            if idx < allLines.Length then
                let lineNo = idx + 1
                yield sprintf "%6d\t%s" lineNo allLines.[idx]
    }
    |> String.concat "\n"

module Fugue.Tools.EditTool

open System
open System.IO
open System.ComponentModel
open Fugue.Tools.PathSafety

[<Description("Edit a text file by exact-string replacement. Requires old_string to be unique unless replace_all is true.")>]
let edit
    ([<Description("Working directory.")>] cwd: string)
    ([<Description("File to modify.")>] path: string)
    ([<Description("Exact text to find (must occur once unless replace_all=true).")>] oldString: string)
    ([<Description("Replacement text.")>] newString: string)
    ([<Description("If true, replace every occurrence; default false.")>] replaceAll: bool option)
    : string =
    let full = resolve cwd path
    if not (File.Exists full) then raise (FileNotFoundException("file not found", full))
    let original = File.ReadAllText full
    let count =
        // count non-overlapping occurrences
        let mutable c, i = 0, 0
        while i >= 0 do
            i <- original.IndexOf(oldString, i, StringComparison.Ordinal)
            if i >= 0 then
                c <- c + 1
                i <- i + max 1 oldString.Length
        c

    if count = 0 then raise (InvalidOperationException $"old_string not found in {full}")
    let all = replaceAll |> Option.defaultValue false
    if count > 1 && not all then
        raise (InvalidOperationException
            $"old_string occurs {count} times in {full}; pass replace_all=true or pick a unique snippet")

    let updated = original.Replace(oldString, newString)
    File.WriteAllText(full, updated)
    sprintf "edited %s (%d replacement%s)" full count (if count = 1 then "" else "s")
```

module Fugue.Tools.BashTool

open System
open System.Diagnostics
open System.Text
open System.ComponentModel

[<Description("Run a shell command via /bin/zsh -lc and return combined stdout/stderr with exit code.")>]
let bash
    ([<Description("Working directory.")>] cwd: string)
    ([<Description("Command line to execute (shell syntax allowed).")>] command: string)
    ([<Description("Timeout in milliseconds (default 120000).")>] timeoutMs: int option)
    : string =
    let timeout = timeoutMs |> Option.defaultValue 120_000

    let psi = ProcessStartInfo("/bin/zsh", "-lc " + "\"" + command.Replace("\"", "\\\"") + "\"")
    psi.WorkingDirectory <- cwd
    psi.RedirectStandardOutput <- true
    psi.RedirectStandardError <- true
    psi.UseShellExecute <- false

    use proc = Process.Start psi
    let stdout = StringBuilder()
    let stderr = StringBuilder()
    proc.OutputDataReceived.Add(fun e -> if not (isNull e.Data) then stdout.AppendLine e.Data |> ignore)
    proc.ErrorDataReceived.Add(fun e -> if not (isNull e.Data) then stderr.AppendLine e.Data |> ignore)
    proc.BeginOutputReadLine()
    proc.BeginErrorReadLine()

    let finished = proc.WaitForExit timeout
    if not finished then
        try proc.Kill(true) with _ -> ()
        sprintf "<timeout after %d ms>\nstdout:\n%s\nstderr:\n%s" timeout (string stdout) (string stderr)
    else
        proc.WaitForExit()  // ensure async readers drain
        sprintf "exit_code=%d\nstdout:\n%s\nstderr:\n%s" proc.ExitCode (string stdout) (string stderr)
```

    ([<Description("Working directory.")>] cwd: string)
    ([<Description("Glob pattern (e.g. **/*.fs).")>] pattern: string)

open System
open System.IO
open System.Text.RegularExpressions
open System.ComponentModel
open Microsoft.Extensions.FileSystemGlobbing
open Fugue.Tools.PathSafety

[<Description("Search files for a regex pattern. Returns matching lines as `path:lineno:text` (one per line).")>]
let grep
    ([<Description("Working directory.")>] cwd: string)
    ([<Description("Regex pattern (.NET regex syntax).")>] pattern: string)
    ([<Description("Search root, defaults to cwd.")>] root: string option)
    ([<Description("Optional include glob (e.g. **/*.fs); default = all files.")>] glob: string option)
    : string =
    let r = root |> Option.map (resolve cwd) |> Option.defaultValue cwd
    let regex = Regex(pattern, RegexOptions.Compiled)

    let files =
        let matcher = Matcher()
        matcher.AddInclude(glob |> Option.defaultValue "**/*") |> ignore
        let res =
            matcher.Execute(
                Microsoft.Extensions.FileSystemGlobbing.Abstractions.DirectoryInfoWrapper(DirectoryInfo r))
        res.Files |> Seq.map (fun f -> Path.Combine(r, f.Path))

    seq {
        for file in files do
            try
                let mutable lineNo = 0
                for line in File.ReadLines file do
                    lineNo <- lineNo + 1
                    if regex.IsMatch line then
                        let rel = Path.GetRelativePath(r, file).Replace('\\', '/')
                        yield sprintf "%s:%d:%s" rel lineNo line
            with _ -> ()  // skip binary / unreadable files
    }
    |> String.concat "\n"

open System
open System.Collections.Generic
open System.Threading
open System.Threading.Tasks
open Microsoft.Extensions.AI
open Anthropic.SDK
open Anthropic.SDK.Messaging

/// Адаптер Anthropic.SDK → Microsoft.Extensions.AI.IChatClient.
/// Поддерживает:
///   - GetResponseAsync (полный ответ)
///   - GetStreamingResponseAsync (стрим текста + function calls)
///   - tool definitions через ChatOptions.Tools
type AnthropicChatClient(apiKey: string, model: string) =
    let client = AnthropicClient(APIAuthentication(apiKey))

    let mapBlocksToMessages (messages: IList<ChatMessage>) : List<Message> =
        let result = List<Message>()
        for m in messages do
            let role =
                match m.Role with
                | r when r = ChatRole.User -> RoleType.User
                | r when r = ChatRole.Assistant -> RoleType.Assistant
                | _ -> RoleType.User // System idет в system-параметр, не в messages
            if m.Role <> ChatRole.System then
                let text =
                    m.Contents
                    |> Seq.choose (function
                        | :? TextContent as t -> Some t.Text
                        | _ -> None)
                    |> String.concat ""
                result.Add(Message(Role = role, Content = text))
        result

    let extractSystem (messages: IList<ChatMessage>) =
        messages
        |> Seq.filter (fun m -> m.Role = ChatRole.System)
        |> Seq.collect (fun m ->
            m.Contents
            |> Seq.choose (function
                | :? TextContent as t -> Some t.Text
                | _ -> None))
        |> String.concat "\n"

    interface IChatClient with
        member _.GetResponseAsync
            (messages: IList<ChatMessage>,
             options: ChatOptions,
             cancellationToken: CancellationToken)
            : Task<ChatResponse> =
            task {
                let systemText = extractSystem messages
                let parameters =
                    MessageParameters(
                        Model = model,
                        Messages = mapBlocksToMessages messages,
                        MaxTokens = (if options.MaxOutputTokens.HasValue then options.MaxOutputTokens.Value else 8192),
                        Stream = false,
                        System =
                            (if String.IsNullOrEmpty systemText then null
                             else List([ SystemMessage(Type = "text", Text = systemText) ])))
                let! resp = client.Messages.GetClaudeMessageAsync(parameters, cancellationToken)
                let text =
                    resp.Content
                    |> Seq.choose (fun block ->
                        if block.Type = "text" then Some block.Text else None)
                    |> String.concat ""
                let chatMsg = ChatMessage(ChatRole.Assistant, text)
                return ChatResponse([| chatMsg |])
            }

        member this.GetStreamingResponseAsync
            (messages: IList<ChatMessage>,
             options: ChatOptions,
             cancellationToken: CancellationToken)
            : IAsyncEnumerable<ChatResponseUpdate> =
            taskSeq {
                let systemText = extractSystem messages
                let parameters =
                    MessageParameters(
                        Model = model,
                        Messages = mapBlocksToMessages messages,
                        MaxTokens = (if options.MaxOutputTokens.HasValue then options.MaxOutputTokens.Value else 8192),
                        Stream = true,
                        System =
                            (if String.IsNullOrEmpty systemText then null
                             else List([ SystemMessage(Type = "text", Text = systemText) ])))
                let stream = client.Messages.StreamClaudeMessageAsync(parameters, cancellationToken)
                for evt in stream do
                    if evt.Delta <> null && not (String.IsNullOrEmpty evt.Delta.Text) then
                        let upd = ChatResponseUpdate()
                        upd.Role <- ChatRole.Assistant
                        upd.Contents <- ResizeArray<AIContent>([| TextContent(evt.Delta.Text) :> AIContent |])
                        yield upd
            }
            |> AsyncEnumerable.ofTaskSeq

        member _.GetService(serviceType: Type, _serviceKey: obj) =
            if serviceType = typeof<AnthropicChatClient> then box this
            else null

        member _.Dispose() = ()

module Fugue.Agent.Discovery

open System
open System.IO
open System.Net.Http
open System.Text.Json
open System.Text.Json.Nodes

type Candidate =
    | EnvKey of provider: string * model: string * keySource: string
    | ClaudeSettings of model: string * apiKey: string
    | OllamaLocal of endpoint: Uri * models: string list

type Sources =
    { ClaudeSettingsPath: string
      OllamaPing: Uri -> Async<string list option> }

let private envCandidate () =
    let env n =
        match Environment.GetEnvironmentVariable n with
        | null | "" -> None
        | v -> Some v
    [ "ANTHROPIC_API_KEY", "anthropic", "claude-opus-4-7"
      "OPENAI_API_KEY", "openai", "gpt-5" ]
    |> List.choose (fun (var, prov, defaultModel) ->
        env var |> Option.map (fun _ -> EnvKey(prov, defaultModel, var)))

let private claudeSettingsCandidate (path: string) =
    if not (File.Exists path) then []
    else
        try
            let json = File.ReadAllText path
            let node = JsonNode.Parse json
            let envObj = node.["env"]
            let key =
                if envObj <> null then
                    match envObj.["ANTHROPIC_API_KEY"] with
                    | null -> None
                    | n -> Some(n.GetValue<string>())
                else None
            let model =
                match node.["model"] with
                | null -> "claude-opus-4-7"
                | n -> n.GetValue<string>()
            match key with
            | Some k -> [ ClaudeSettings(model, k) ]
            | None -> []
        with _ -> []

/// Default Ollama ping using HttpClient (suitable for production; tests inject a fake).
let httpOllamaPing (timeoutMs: int) (endpoint: Uri) : Async<string list option> =
    async {
        try
            use http = new HttpClient(Timeout = TimeSpan.FromMilliseconds(float timeoutMs))
            let! resp = http.GetAsync(Uri(endpoint, "/api/tags")) |> Async.AwaitTask
            if not resp.IsSuccessStatusCode then return None
            else
                let! body = resp.Content.ReadAsStringAsync() |> Async.AwaitTask
                let node = JsonNode.Parse body
                let arr = node.["models"] :?> JsonArray
                let names =
                    arr
                    |> Seq.cast<JsonNode>
                    |> Seq.map (fun m -> m.["name"].GetValue<string>())
                    |> List.ofSeq
                return Some names
        with _ -> return None
    }

let discover (sources: Sources) : Candidate list =
    let env = envCandidate ()
    let claude = claudeSettingsCandidate sources.ClaudeSettingsPath
    let ollamaEndpoint = Uri "http://localhost:11434"
    let ollama =
        sources.OllamaPing ollamaEndpoint
        |> Async.RunSynchronously
        |> Option.map (fun models -> [ OllamaLocal(ollamaEndpoint, models) ])
        |> Option.defaultValue []
    env @ claude @ ollama

let defaultSources () : Sources =
    let home =
        match Environment.GetEnvironmentVariable "HOME" with
        | null | "" -> Environment.GetEnvironmentVariable "USERPROFILE"
        | h -> h
    { ClaudeSettingsPath = Path.Combine(home, ".claude", "settings.json")
      OllamaPing = httpOllamaPing 200 }

/// Render numeric menu, read user choice from stdin (1-based or 'q').
let prompt (candidates: Candidate list) : Candidate option =
    if List.isEmpty candidates then None
    else
        eprintfn ""
        eprintfn "No ~/.fugue/config.json found. I detected:"
        candidates
        |> List.iteri (fun i c ->
            let label =
                match c with
                | EnvKey(p, m, src) -> sprintf "%s — using %s from env (model: %s)" p src m
                | ClaudeSettings(m, _) -> sprintf "Anthropic — reuse Claude Code config (model: %s)" m
                | OllamaLocal(ep, models) -> sprintf "Ollama — %A (%d models available)" ep (List.length models)
            eprintfn "  [%d] %s" (i + 1) label)
        eprintf "\nPick one [1-%d], or 'q' to quit: " candidates.Length
        let line = Console.ReadLine()
        match line with
        | null | "q" | "Q" -> None
        | s ->
            match Int32.TryParse s with
            | true, n when n >= 1 && n <= candidates.Length -> Some candidates.[n - 1]
            | _ -> None
```

module Fugue.Agent.Conversation

open System
open System.Collections.Generic
open System.Threading
open System.Threading.Tasks
open Microsoft.Agents.AI
open Microsoft.Extensions.AI
open FSharp.Control

/// UI-friendly conversation events.
type Event =
    | TextChunk of string
    | ToolStarted of id: string * name: string * argsJson: string
    | ToolCompleted of id: string * output: string * isError: bool
    | Finished
    | Failed of exn

/// Run a single user-input through the AIAgent and surface UI events.
/// `thread` is the persistent AgentSession (None — start fresh).
/// MAF crutит весь tool-loop сама; мы просто наблюдаем и трансформируем.
let run
    (agent: AIAgent)
    (thread: AgentSession option)
    (userInput: string)
    (cancel: CancellationToken)
    : IAsyncEnumerable<Event> =

    taskSeq {
        let t = thread |> Option.defaultWith (fun () -> agent.GetNewThread())
        let mutable failed = false
        try
            let stream = agent.RunStreamingAsync(userInput, t, cancellationToken = cancel)
            let enumerator = stream.GetAsyncEnumerator(cancel)
            let mutable hasNext = true
            while hasNext do
                let! step = enumerator.MoveNextAsync().AsTask()
                if step then
                    let upd = enumerator.Current
                    if not (isNull upd.Contents) then
                        for c in upd.Contents do
                            match c with
                            | :? TextContent as tc when not (String.IsNullOrEmpty tc.Text) ->
                                yield TextChunk tc.Text
                            | :? FunctionCallContent as fc ->
                                let argsJson =
                                    try System.Text.Json.JsonSerializer.Serialize fc.Arguments
                                    with _ -> "{}"
                                yield ToolStarted(fc.CallId, fc.Name, argsJson)
                            | :? FunctionResultContent as fr ->
                                let output = if isNull fr.Result then "" else string fr.Result
                                let isError =
                                    fr.Exception <> null
                                yield ToolCompleted(fr.CallId, output, isError)
                            | _ -> ()
                else
                    hasNext <- false
        with ex ->
            failed <- true
            yield Failed ex
        if not failed then yield Finished
    }
```

```fsharp
module Fugue.Cli.Mvu

open System
open System.Threading
open Terminal.Gui

/// Effects (Cmd) — побочные действия, которые dispatched-аются обратно в update.
type Cmd<'msg> = (('msg -> unit) -> unit) list

module Cmd =
    let none<'msg> : Cmd<'msg> = []
    let ofMsg (m: 'msg) : Cmd<'msg> = [ fun dispatch -> dispatch m ]
    let ofSub (sub: ('msg -> unit) -> unit) : Cmd<'msg> = [ sub ]
    let batch (cmds: Cmd<'msg> list) : Cmd<'msg> = List.concat cmds

/// View-tree — описание UI как данных. Diff'инг ленивый: при каждом render'е мы пересоздаём
/// детей корневого Window'а. Для MVP это работает: рендер текстов и tool-блоков дёшев.
type ViewNode =
    | Window of title: string * children: ViewNode list
    | Frame of title: string * x: int * y: int * w: Dim * h: Dim * children: ViewNode list
    | Label of x: int * y: int * text: string * color: Color option
    | TextField of x: int * y: int * w: Dim * value: string * onChange: (string -> unit)
    | StatusBar of items: (Key * string * (unit -> unit)) list

and Dim =
    | DAt of int
    | DFill
    | DPercent of int

let private toTGDim (d: Dim) =
    match d with
    | DAt n -> Terminal.Gui.Dim.Absolute(n)
    | DFill -> Terminal.Gui.Dim.Fill()
    | DPercent p -> Terminal.Gui.Dim.Percent(float32 p)

let rec private build (node: ViewNode) : View =
    match node with
    | Window (title, kids) ->
        let w = new Window()
        w.Title <- title
        for child in kids do w.Add(build child)
        w :> View
    | Frame (title, x, y, w, h, kids) ->
        let f = new FrameView()
        f.Title <- title
        f.X <- Pos.At x
        f.Y <- Pos.At y
        f.Width <- toTGDim w
        f.Height <- toTGDim h
        for child in kids do f.Add(build child)
        f :> View
    | Label (x, y, text, color) ->
        let l = new Label()
        l.X <- Pos.At x
        l.Y <- Pos.At y
        l.Text <- text
        match color with
        | Some c -> l.ColorScheme <- ColorScheme(Normal = Attribute(c, Color.Black))
        | None -> ()
        l :> View
    | TextField (x, y, w, value, onChange) ->
        let tf = new TextField()
        tf.X <- Pos.At x
        tf.Y <- Pos.At y
        tf.Width <- toTGDim w
        tf.Text <- value
        tf.add_TextChanged (fun _ -> onChange (string tf.Text))
        tf :> View
    | StatusBar items ->
        let sb =
            new StatusBar(
                items
                |> List.map (fun (k, label, action) ->
                    Shortcut(k, label, Action(fun () -> action())))
                |> Array.ofList)
        sb :> View

/// Run an MVU program. Pure update (model, msg → model * cmd) + view (model → ViewNode).
let run<'model, 'msg>
    (init: unit -> 'model * Cmd<'msg>)
    (update: 'msg -> 'model -> 'model * Cmd<'msg>)
    (view: 'model -> ('msg -> unit) -> ViewNode)
    : unit =

    Application.Init()
    try
        let mutable model, initCmd = init ()
        let dispatchRef : ref<('msg -> unit)> = ref (fun _ -> ())
        let root = build (view model dispatchRef.Value)
        Application.Top.Add root

        let render () =
            Application.Top.RemoveAll()
            let v = build (view model dispatchRef.Value)
            Application.Top.Add v
            Application.Top.SetNeedsDisplay()

        let dispatch (msg: 'msg) =
            let model', cmd = update msg model
            model <- model'
            render ()
            for sub in cmd do
                Application.MainLoop.Invoke(fun () -> sub dispatchRef.Value)

        dispatchRef := dispatch
        for sub in initCmd do sub dispatchRef.Value

        Application.Run(Application.Top)
    finally
        Application.Shutdown()
```

> **Замечания:**
> - Это **минимальный** MVU. Полный diff'инг дочерних views — **не делаем**: при каждом dispatch'е пересоздаём детей корневого Top. Для MVP с 30–100 блоками в чате это приемлемо. Если профиль покажет фризы — добавим diff в v0.2.
> - Точные имена API Terminal.Gui v2 (`Pos.At`, `Dim.Absolute`, `add_TextChanged`, `Shortcut`-конструктор `StatusBar`) могут отличаться по minor-версиям. Сверяй с XML-док пакета.
> - `Application.MainLoop.Invoke` — гарантирует, что dispatch выполняется в UI-thread'е. Без него async Cmd'ы (стрим из MAF) поломают рендер.

module Fugue.Cli.Update

open System.Text
open System.Threading
open Fugue.Cli.Model

let private appendBlock (b: Block) (history: Block list) = history @ [ b ]

let private updateLastBlock (history: Block list) (f: Block -> Block) : Block list =
    match List.tryLast history with
    | None -> history
    | Some _ ->
        let init = history |> List.take (List.length history - 1)
        init @ [ f (List.last history) ]

/// Pure update — no Cmd dispatching here. Effects (streaming) wired in Program.fs via Cmd.ofSub.
let update (msg: Msg) (m: Model) : Model * (unit -> unit) option =
    match msg with
    | InputChanged s -> { m with Input = s }, None

    | Submit when m.Streaming || m.Input.Trim() = "" -> m, None
    | Submit ->
        let userBlock = UserMsg m.Input
        let updated =
            { m with
                History = appendBlock userBlock m.History
                Input = ""
                Streaming = true
                Status = "thinking…" }
        updated, None  // Effect wired by Program.fs (Cmd.ofSub)

    | TextChunk t ->
        let history' =
            match List.tryLast m.History with
            | Some (AssistantText sb) ->
                sb.Append t |> ignore
                m.History  // mutated in place
            | _ ->
                let sb = StringBuilder(t)
                appendBlock (AssistantText sb) m.History
        { m with History = history' }, None

    | ToolStarted(id, name, args) ->
        { m with History = appendBlock (ToolBlock(id, name, args, Running)) m.History }, None

    | ToolCompleted(id, output, isError) ->
        let history' =
            m.History
            |> List.map (fun b ->
                match b with
                | ToolBlock(bid, name, args, _) when bid = id ->
                    let st = if isError then FailedTool output else Done output
                    ToolBlock(bid, name, args, st)
                | other -> other)
        { m with History = history' }, None

    | StreamFinished ->
        { m with Streaming = false; Cancel = None; Status = "" }, None

    | StreamFailed err ->
        { m with
            Streaming = false
            Cancel = None
            Status = ""
            History = appendBlock (ErrorBlock err) m.History }, None

    | Cancel ->
        m.Cancel |> Option.iter (fun cts -> try cts.Cancel() with _ -> ())
        { m with Streaming = false; Cancel = None; Status = "cancelled" }, None

    | Quit ->
        Terminal.Gui.Application.RequestStop()
        m, None
```

    match p with
    | PAt n -> Terminal.Gui.Pos.At(n)
    | PPercent p -> Terminal.Gui.Pos.Percent(float32 p)
    | PBottom n -> Terminal.Gui.Pos.AnchorEnd(n)
```

И измени `Frame` в `ViewNode`:

```fsharp
| Frame of title: string * x: Pos * y: Pos * w: Dim * h: Dim * children: ViewNode list
```

В `build` соответственно — `f.X <- toTGPos x; f.Y <- toTGPos y`.

```fsharp
module Fugue.Cli.Program

open System
open System.Threading
open Microsoft.Agents.AI
open Microsoft.Extensions.AI
open Terminal.Gui
open Fugue.Core.Config
open Fugue.Core.SystemPrompt
open Fugue.Agent
open Fugue.Cli.Model
open Fugue.Cli.Mvu
open Fugue.Cli.Update
open Fugue.Cli.View

let private buildAgent (cfg: AppConfig) : AIAgent =
    let cwd = Environment.CurrentDirectory
    let tools = Fugue.Tools.ToolRegistry.buildAll cwd
    let sysPrompt =
        match cfg.SystemPrompt with
        | Some s -> s
        | None -> SystemPrompt.render cwd Fugue.Tools.ToolRegistry.names
    AgentFactory.create cfg.Provider sysPrompt tools

/// Cmd, который запускает стрим Conversation и dispatches события в Update.
let private streamCmd (agent: AIAgent) (thread: AgentSession option) (userInput: string) : Cmd<Msg> =
    [ fun dispatch ->
        let cts = new CancellationTokenSource()
        // Сохраняем CTS в Model через специальное сообщение
        dispatch (StreamStarted cts)
        Async.Start(async {
            try
                let stream = Conversation.run agent thread userInput cts.Token
                let enumerator = stream.GetAsyncEnumerator(cts.Token)
                let mutable hasNext = true
                while hasNext do
                    let! step =
                        enumerator.MoveNextAsync().AsTask() |> Async.AwaitTask
                    if step then
                        match enumerator.Current with
                        | Conversation.TextChunk s -> dispatch (TextChunk s)
                        | Conversation.ToolStarted(id, n, a) -> dispatch (ToolStarted(id, n, a))
                        | Conversation.ToolCompleted(id, o, e) -> dispatch (ToolCompleted(id, o, e))
                        | Conversation.Finished -> dispatch StreamFinished
                        | Conversation.Failed ex -> dispatch (StreamFailed ex.Message)
                    else
                        hasNext <- false
            with
            | :? OperationCanceledException -> dispatch (StreamFailed "cancelled")
            | ex -> dispatch (StreamFailed ex.Message)
        }, cts.Token)
    ]

[<EntryPoint>]
let main argv =
    match Config.load argv with
    | Error (NoConfigFound help) ->
        let candidates = Discovery.discover (Discovery.defaultSources ())
        match Discovery.prompt candidates with
        | Some chosen ->
            let provider =
                match chosen with
                | Discovery.EnvKey("anthropic", model, _) ->
                    Anthropic(Environment.GetEnvironmentVariable "ANTHROPIC_API_KEY", model)
                | Discovery.EnvKey("openai", model, _) ->
                    OpenAI(Environment.GetEnvironmentVariable "OPENAI_API_KEY", model)
                | Discovery.ClaudeSettings(model, key) ->
                    Anthropic(key, model)
                | Discovery.OllamaLocal(ep, models) ->
                    let m = if List.isEmpty models then "llama3.1" else List.head models
                    Ollama(ep, m)
                | _ -> failwith "unsupported candidate"
            let cfg = { Provider = provider; SystemPrompt = None; MaxIterations = 30 }
            Config.saveToFile cfg
            runApp cfg
        | None ->
            eprintfn "%s" help
            1
    | Error (InvalidConfig reason) ->
        eprintfn "Invalid config: %s" reason
        1
    | Ok cfg -> runApp cfg

and runApp (cfg: AppConfig) : int =
    let agent = buildAgent cfg

    let init () : Model * Cmd<Msg> =
        { Config = cfg
          Thread = None
          History = []
          Input = ""
          Streaming = false
          Cancel = None
          Status = "" }, Cmd.none

    let updateWithEffects (msg: Msg) (model: Model) : Model * Cmd<Msg> =
        let m', _ = Update.update msg model
        match msg with
        | Submit when m'.Streaming ->
            // Берём userInput из последнего блока истории
            let userInput =
                match List.tryLast m'.History with
                | Some (UserMsg s) -> s
                | _ -> ""
            // Создаём thread лениво: первая отправка делает GetNewThread()
            let thread =
                match m.Thread with
                | Some t -> t
                | None -> agent.GetNewThread()
            { m' with Thread = Some thread }, streamCmd agent (Some thread) userInput
        | _ -> m', Cmd.none

    run init updateWithEffects view
    0
```

> **Замечания для разработчика:**
> 1. **`Msg.StreamStarted of CancellationTokenSource`** — добавь этот case в `Model.Msg` и обработай в `Update.update`: запиши `cts` в `model.Cancel`. Это ~3 строки в `Update.fs` (Task 23). Цель — UI знает, какой CTS отменить при Esc.
> 2. **Threading.** `Async.Start` диспатчит сообщения **из background-thread'а**, а Terminal.Gui v2 принимает изменения только в UI-thread. В `Mvu.fs` (Task 21B) `dispatch` уже оборачивает каждый Cmd в `Application.MainLoop.Invoke`. Если в твоей версии TG v2 главный loop называется иначе (`Application.Invoke`?) — поправь.
> 3. **`agent.GetNewThread()`** — точное имя зависит от версии MAF. В 1.0 это, скорее всего, `agent.GetNewThread()` или `new ChatHistoryAgentSession()`. Сверь.
