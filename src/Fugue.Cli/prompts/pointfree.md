command: pointfree
args: [fn]
description: Offer point-free rewrite of an F# function
---
Offer a point-free rewrite of the following F# function:

```fsharp
{fn}
```

Format your answer as:
1. **Original** — repeat the function as-is.
2. **Point-free equivalent** — the rewritten version using partial application, `>>` / `<<`, or combinators.
3. **Transformation steps** — explain each step of the rewrite.
4. **Readability verdict** — note whether the point-free form is actually clearer, and when to prefer the explicit form.

If a point-free form is not natural or readable, say so and explain why.
